How it works

Up and running in seconds

Install once. Your speed follows you everywhere.

1

Install

Add PlayFaster to Chrome from the Web Store. Free, one click.

2

Open the popup

Click the extension icon on any page with video or audio.

3

Use < > buttons

Tap < / > to step by 0.1x, or << / >> to jump by 0.5x.

4

Type or scroll

Enter any value from 0.1 to 16 directly, or scroll your mouse wheel to nudge the speed.

5

It remembers

Your preference is saved automatically — same speed, every page, every restart.
